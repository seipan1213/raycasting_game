# raycasting_game for linux
## 操作方法
移動　WASD 方向　←→   ESC or Xボタンで終了
## コンパイル
make
## 実行
./cub3D test.cub 
## BMPファイルを生成
./cub3D test.cub --save  

## 設定ファイル　
SO　南のテクスチャ

WE　西のテクスチャ

NO　北のテクスチャ

EA　東のテクスチャ

S　スプライト

R　画面サイズ

C　空の色

F　床の色


Map　0　なし　1  壁　2　スプライト　NSWE プレイヤーと方角
